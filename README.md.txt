# 🎰 NEON HEIST - iGaming Slot Demo

**6×5 Cluster Pays + Cascade**  
**Smooth Reel Spin**  
**HTML5 + CSS3 + Vanilla JavaScript**

Professional frontend slot machine demo built for portfolio purposes.

### ✨ Features

- Full 6×5 grid with cluster pays mechanic
- Smooth reel spinning with stagger effect
- Win calculation and balance system
- Clean neon iGaming visual style
- Paytable showing multipliers and potential wins
- Auto Spin functionality
- Fully responsive

### 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox + CSS Animations)
- Vanilla JavaScript
